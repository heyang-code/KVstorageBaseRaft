# KVstorageBaseRaft
## cpp实现的基于Raft算法的k-v存储数据库
本项目主要是学习Raft的原理，并用muduo和rpc框架实现的一个基于raft算法的简单k-v存储数据库,主要用于观察选举过程  
## 项目编译运行
1. 先创建项目编译目录 mkdir cmake-build-debug
2. cd cmake-build-debug
3. cmake ..
4. make
