# KVstorageBaseRaft
# cpp实现的基于Raft算法的k-v存储数据库
本项目主要是学习Raft的原理，并实现一个简单的k-v存储数据库
# 分支说明
main：实现一个简单的clerk  
rpc：muduo和rpc框架相关内容  
raft_DB：基于Raft的k-v存储数据库，主要用于观察选举过程  
# 项目文档目录说明
├── bin 生成的可执行文件存放地  
├── cmake-build-debug 项目编译目录，默认是没有的，需要自己创建 
├── docs    项目文档存放地
├── bin 生成的可执行文件存放地  
├── cmake-build-debug 项目编译目录，默认是没有的，需要自己创建
